# Scripts

Automation helpers for maintaining the learning journal and portfolio.

## Purpose

Automate repetitive tasks:
- Calculate days since journey start
- Generate project demo pages
- Deploy to GitHub Pages
- Update portfolio statistics

## Available Scripts

- `update-day-counter.sh` - Auto-calculate days since first commit
- `generate-demo-page.py` - Generate demo page from project data (future)
- `deploy.sh` - Deploy portfolio to GitHub Pages (future)

## Usage

Run scripts from the repository root:

```bash
./scripts/update-day-counter.sh
```

Add new scripts as automation opportunities emerge.
