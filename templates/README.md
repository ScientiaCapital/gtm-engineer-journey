# Templates

Consistency helpers for maintaining the portfolio and learning journal.

## Purpose

Templates make it easy to:
- Add new projects to the portfolio
- Generate consistent demo pages
- Document milestones in a standard format
- Keep learning logs structured

## Available Templates

- `project-entry.json` - Template for adding projects to `docs/projects.json`
- `demo-page-template.html` - Template for creating project demo pages
- `milestone-template.md` - Template for milestone documentation
- `learning-log-template.md` - Template for weekly learning logs

## Usage

1. Copy the template
2. Fill in your content
3. Follow the structure for consistency

These aren't rigid - adapt as needed, but keep the core structure intact.
