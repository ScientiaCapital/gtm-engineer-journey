# Resources

## Purpose

Document the learning path:
- Books that influenced thinking
- Courses and bootcamps completed
- Tools and tech stack evolution
- Thought leaders worth following

This shows **intellectual influences** and provides a resource guide for others on similar journeys.

## Files

- `books.md` - Books that shaped GTM + technical thinking
- `tools.md` - Tech stack, frameworks, platforms
- `courses.md` - Bootcamps, tutorials, certifications (optional)
- `people-to-follow.md` - Thought leaders in GTM x AI space (optional)

## Usage

Update as you discover valuable resources. Add notes about why each resource matters.
